LokSuraksha App

Professional Summary#

Developed LokSuraksha — a comprehensive citizen safety platform that empowers users to report community issues (like potholes, hazards, and safety concerns) using a mobile app with camera and GPS integration. The system features:

  • A React Native (Expo) mobile app for capturing and sending reports
  • A Node.js + Express backend API with secure authenticated admin controls
  • A React + Vite web admin dashboard to manage, filter, and visualize reports
  • Seamless data persistence via SQLite, and a shared TypeScript workspace for types & API clients

This end-to-end solution improves public issue reporting and tracking with offline support, responsive UI, and modular architecture.

Key Features#

  • Mobile app with camera and GPS integration
  • Secure admin authentication system
  • Real-time report tracking
  • Offline support with data queueing
  • Responsive web admin dashboard
  • Type-safe development with shared TypeScript workspace

Technology Stack#

  • Frontend Mobile: React Native, Expo
  • Frontend Web: React.js, Vite
  • Backend: Node.js, Express.js
  • Database: SQLite
  • Languages: TypeScript, JavaScript
  • Other: Docker, HTML, CSS, Secure Shell (SSH)
