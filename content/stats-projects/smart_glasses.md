## Overview

Developed smart glasses for visually impaired individuals using ESP32, integrating AI-powered object detection, obstacle avoidance, and voice assistance. Implemented real-time image processing and text-to-speech conversion to enhance navigation and accessibility.

## Key Features

- AI-powered object detection
- Obstacle avoidance system
- Voice assistance for navigation
- Real-time image processing
- Text-to-speech conversion
- Enhanced accessibility for visually impaired users

## Technology Stack

- **Hardware**: ESP32
- **AI/CV**: Artificial Intelligence & Computer Vision
- **IoT**: IoT & Voice Assistance
- **Other**: Embedded Systems
