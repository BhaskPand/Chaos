## Overview

Built a full-stack AI-integrated web application as part of the Full Stack Developer Assessment by Bhatiyani Astute Intelligence Pvt. Ltd.

## Tech Stack

- **Frontend**: React.js with MUI (Material UI)
- **Backend**: Flask (Python), SQLite
- **Deployment**: Vercel (frontend), Render (backend)
- **AI Integration**: OpenAI GPT-3.5 API

## Key Features

- Add/delete and manage custom exams
- Visual exam trends using charts
- Generate personalized study plans using OpenAI
- Clean UI/UX and mobile responsiveness
- API environment configuration using .env

## Learnings

- Hands-on experience building and deploying full-stack applications
- API integration and backend database management
- Environment variable handling and Git workflow
- Problem-solving and debugging in live deployment

## Skills

React.js, MUI, Flask, SQLite, OpenAI, Render, Vercel
